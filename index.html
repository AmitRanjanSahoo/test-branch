Change Default Branch


Here is the description of how to change your created branch
to default branch.

In GitHub, everything stored like in your computer stores the files, inside a folder. Here, instead of folders, it has branches, each branch can have files and resourses of their own. Each branch is also part of a respiratory, means, that respiratory can have as many branches as it requires. Tink as there is an industry(respiratory), and it has many factories(branches).

Create a new branch

  • Go to the GitHub site.
  • Login to your profile.
  • Go to the create button at the top right side of your display, you'll find a circular symbol to create new respiratory.
    or
    You can go to existing respiratory.
  • You'll find a dropdown at leftside of display, just above the respiratory files. Click that.
  • If not available create a new branch from there.
  • Go to settings on the same line from where branch drop down was present.
  • In the general settings you will find Default branch.
  • There is arrow like button present, click that.
  • There choose the branch you want to make default for that respiratory. Then click Update.
    • Now, you're done. That branch got selected as default!

You can set rules for branches. Change the structure and many more. Main is set as default but it's not constant!
